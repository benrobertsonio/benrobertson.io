## Local Dev

 1. Clone the repo.
 1. `nvm use`
 1. `npm install`
 2. `npm start` to run Gatsby locally.

## Deployment

Branches deploy automatically to [Gatsby Cloud](https://www.gatsbyjs.com/get-started)!
